# CNV Comparison Tool

A Python tool for comparing CNV (Copy Number Variation) data from different sources:
- GenomStudio
- NxClinical
- Custom Software
- Gold Standard

## Usage

```bash
python cnv_comparator.py <GS_File> <NX_File> <OM_File> <Gold_File> <Output_File>
```

## Requirements
- Python 3.x
- pandas
- openpyxl
